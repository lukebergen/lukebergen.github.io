Nest

References

Apologies in advance. I've tried to keep all the philosophy stuff in its own little section of this wiki. But this topic reeeally pulls me towards that stuff. I'll try to be diligent about keeping things practical.

There are 16 words that all follow a common pattern. They're the two letter words that start with a consonant and end in "y". So py, by, ky, etc... These are the "reference" words. The definitions for these are 

py = the "p" reference
by = the "b" reference
etc...

That's not especially helpful. Hence this entire page to talk about what it means for something to be "the 'p' reference".

ny is a special case so we'll set it aside for now while we address the other 15.

Reference words are most similar to pronouns of other languages but are a little different in some interesting ways.

Unused, reference words refer to anything/everything that can be expressed. They take on meaning in a conversation by how they're used. Let me use dy in an example conversation to show you what I mean. But first, real quick... 

co x = $x but decorated with the meta-information that the speaker wants to know more about it (e.g. `co kleso do ti` => "do you like it?", "how do you like it?", etc...)
  • Alice: Let's have co dy for dinner (what do you want to do for dinner (we'll use dy to refer to whatever it is))
  • Bob: dy isn't pizza. We just had that
  • Alice: Agreed. dy isn't too heavy. I'm not super hungry
  • Bob: could dy be sushi?
  • Alice: dy is sushi or maybe tacos.
  • Bob: Ooo tacos. dy is definitely tacos.

Think of dy in this conversation as a circle in a Venn diagram. Before being used at all, it's a circle that encompasses the entire diagram. When Alice uses it in the first sentence, the circle narrow significantly down to only those things that would make sense in the sentence "Let's have <it> for dinner". The circle gets narrower and narrower before finally becoming the relatively small circle "tacos". Though, note: dy is still a circle here. It can continue being narrowed infinitely finer and finer and will never become a singular point (aaaahh stop. Leave it for philosophy).

Generally, the more a reference gets used throughout a conversation, the narrower it's possible meaning becomes. The fact that we have so many of these references and that they start with each of the 16 consonants allows for some fun uses. A couple quick definitions... 

te x = 0 $x
ca x = 1 $x
go x = 2 $x
bu x = 3 $x
vi x = 4 $x
je x = 5 $x
ra x = 6 $x
mo x = 7 $x
su x = 8 $x
di x = 9 $x
  • Alice: you want co cy sugars in your coffee (how many sugars do you want (we'll call the number cy))
  • Bob: sigh. ca sugar

Bob can reply with any number he wants. But by selecting the cy reference word to ask Bob which number he'd like, Alice lightly hints that maybe a "c" number word might be what Bob is looking for. Another example.

  • Alice: I saw Paul and Greg at the store earlier
  • Bob: Are they together?
  • Alice: nah, py is seeing ty remember?
  • Bob: co ty?
  • Alice: ty is Tim

This leads to another helpful set of words. For each of the reference words, there's also a word that ends with "i". So pyi, byi, cyi, etc... 

pyi x = "py" is $x. $x
byi x = "by" is $x. $x
etc...

pyi works exactly the same way that py does except that it takes in a single value and starts the py reference off as starting with "and it starts with that". So our earlier example talking about dinner could have been started with

  • Alice: let's eat co dyi dinner

All Englishized like this it might not seem as clear where this would be useful, but in pure Nest, it can be a handy shortcut.

Ok, we've put it off long enough. ny. In Nest, if a word requires 3 values, then 3 values it must have. A sentence that only gives 2 values to a 3-value word is ungrammatical and would leave a listener waiting / expecting more to come.

But what if I don't want/need to specify one of the arguments?

  • Alice: would you like to make a donation?
  • Bob: sonda mi ny ny (here we're trying to say "I already gave". The how much and the who-to are irrelevant or discernible from context)

So ny can be thought of as "grammatically I'm forced to say something here, but I wish I could leave it blank". Dipping dangerously close to philosophy, in our Venn diagram analogy from earlier, ny represents a special circle that always encompasses the entire diagram and can never be narrowed further. This way, it always represents "anything could be said here and be a legitimate interpretation".

So technically, Bob would be telling the truth even if he had given nothing since "0" is a legitimate value that ny could represent. But in practice, that'd be a dick move and nobody would assume that's what was meant. That might sound hand-wavey. And it is. But we do do this hand-waviness thoughtfully. We'll talk more about this "you know what I mean" type stuff in context where these reference words feature heavily.
