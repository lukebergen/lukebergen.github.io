vim notes

vimwiki

  • All of these (except <Leader>w<Leader>i) should take a leading number in order to open a particular wiki (e.g. 2<Leader>ww to open personal wiki)
  • Opening wiki
    • <Leader>ww -- Open wiki index
    • <Leader>wt -- Open wiki index in new tab
  • Diary
    • <Leader>wi -- Open diary index
    • <Leader>w<Leader>i -- On diary index page, regenerate index
    • <Leader>w<Leader>w -- Open diary entry for today
    • <Leader>w<Leader>t -- Open diary entry for today in a new tab
  • Lists
    • <C-Space> -- Toggle checkbox of list item on/off (creates checkbox if one not present in list item)
    • gl* -- Make current line a list item using "*". Sub "-", "#", "1", "a", "i", "I"
    • gL* -- Change symbol of current list to "*". Sub "-", "#", "1", "a", "i", "I"
  • Misc
    • <Tab> -- Next link
    • <Shift-Tab> -- Previous link
    • <CR> -- Create link for selected word an follow it
    • + -- Make selected word a link but do not follow it
    • Generate HTML for wiki: Vimwiki2HTML or VimwikiAll2HTML to convert everything
